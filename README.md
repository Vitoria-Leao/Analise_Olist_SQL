# Analise_Olist_SQL
Este é um repositório de consultas SQL utilizadas para realizar uma análise de dados do dataset público de e-commerce disponibilizado pela Olist.

## Objetivo

Analisar dados do e-commerce Olist para identificar padrões de vendas e clientes mais valiosos utilizando SQL.

## Dataset

Brazilian E-Commerce Public Dataset by Olist.

## Ferramentas
- SQLite;
- SQL;
- GitHub.

## Perguntas Respondidas
- Como a receita evolui?
- Quem são os principais clientes?
- Quais estados geram mais receita?
- Quais produtos lideram cada categoria?
- Existe crescimento mensal?

## Principais Conceitos Aplicados
- SQL Básico: SELECT, JOIN, GROUP BY;
- SQL Avançado: Views, CTEs, Window Functions.

## Principais Insights
- Um quarto do total de clientes é responsável por quase 60% da receita;
- Há uma tendência média de crescimento de 30% na receita acumulada anual desde 2016 até 2018;
- O estado de São Paulo concentra mais de 37% da receita total;
- De forma geral, a receita mensal é maior durante o inverno e o outono.
